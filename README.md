# France 2026

Persönliche Reisewebsite für den Familien-Roadtrip nach Frankreich im Sommer 2026.

## Live-Nutzung

Die Website ist für GitHub Pages gebaut und funktioniert als statische Seite ohne Backend.

## Features

- Blogartige Startseite
- Route mit drei Hauptstationen
- Tagesplaner für 20.07.–07.08.2026
- Filter nach Stationen
- Ausflugs- und Restaurantkarten
- Google-Maps-Suchlinks
- Kinderbereich mit Aktivitätsauswahl
- Abhakbare Packlisten mit Speicherung im Browser
- Einfaches Urlaubstagebuch mit lokaler Speicherung
- Responsives Design für iPhone und Desktop

## Struktur

- `index.html` – Seitenstruktur
- `styles.css` – Gestaltung und Responsive Layout
- `data.js` – Reiseinhalte, Tagesdaten, Orte, Packlisten
- `app.js` – Filter, Checklisten, Tagebuch, Navigation

## Pflege

Die meisten Inhalte stehen in `data.js`. Neue Orte, Tage, Restaurants oder Packlisten können dort ergänzt werden.

## Route

- Darmstadt → Tours
- Tours → Moliets-et-Maâ
- Moliets-et-Maâ → Le Tholonet
- Le Tholonet → Zwischenstopp → Darmstadt
