# France 2026

Eine kleine statische Website für den Familien-Roadtrip nach Frankreich im Sommer 2026.

## Inhalt

- `index.html` – Blogartige Urlaubsseite mit Route, Stationen und Ideen
- `styles.css` – Responsives Styling mit Reiseblog-Optik

## Route

- Darmstadt → Tours
- Tours → Moliets-et-Maâ
- Moliets-et-Maâ → Le Tholonet
- Le Tholonet → Zwischenstopp → Darmstadt

## Lokale Vorschau

Einfach `index.html` im Browser öffnen.

## GitHub Pages

Wenn GitHub Pages aktiviert wird, eignet sich die Seite direkt für Deployment über den Branch `main` und den Root-Ordner.
